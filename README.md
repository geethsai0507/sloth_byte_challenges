# sloth_byte_challenges
my solutions

## Simple Algebra Evaluator
This Python script (**Evaluating_Simple_Algebra.py**) solves simple algebraic equations of the form "x + N = M", "N + x = M", "x - N = M", "N - x = M" for x. 
Run the script and enter an equation when prompted to get the value of x

## Remove Last Vowel
This Python script (**remove_last_vowel.py**) removes the last vowel from each word in a string.
Run the script and enter a string when prompted to get the result with the last vowel removed from each word
